# discriminaton

В рамках проекта мы пользовались данными скриптами.
Большое количество вычислений сделано непосредственно с помощью Google Spreadsheets:
https://docs.google.com/spreadsheets/d/1cI88STHgGswfG0r_Z-4cWF_UTdyrczgG5V-ayIz1Wa4/edit?usp=sharing

Базой для материала стали материалы расследования дата-журналистки РБК Дады Линделл.
В рамках полученных данных мы использовали библиотеку ```russiannames``` (https://github.com/datacoon/russiannames, Иван Бегтин, "Инфокультура") для определения и проверки пола депутатов.

Для формирования конечного материала использовались infogram, datawrapper и tilda.
Сайт доступен по ссылке: http://project1306389.tilda.ws/
